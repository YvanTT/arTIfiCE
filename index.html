arTIfiCE is a jailbreak for TI CE calculators with OS 5.5 / 5.6 / 5.7 / 5.8.
It brings back ASM programs and games!

Compatibility

arTIfiCE works on all CE calculators:
  • TI-83 Premium CE (Edition Python)
    OS 5.5, 5.6, 5.7, 5.8
  • TI-84 Plus CE (Python)
    OS 5.5, 5.6, 5.7, 5.8
  • TI-84 Plus CE-T (Python Edition)
    OS 5.5, 5.6, 5.7, 5.8
  • TI-82 Advanced Edition Python (via arTIfi82)
    OS 5.6.3, 5.6.4
Note: arTIfiCE is not needed if your calculator has OS 5.4 or earlier!
